[PLAYBOOK] Surto de Ransomware

Ordem cronológica de mitigação agressiva de ameaças ativas na rede interna.

  • Isolamento Físico: Desconectar switches afetados e isolar vlans.
  • Bloqueio de C2: Inserir IPs e domínios maliciosos identificados no Firewall.
  • Coleta de Evidências: Dump de memória RAM (Volatility) antes do shutdown.
  • Revogação de Credenciais: Reset forçado de senhas de contas comprometidas.

[AUTOMAÇÃO] Log Triage Tool

Script rápido em Python para identificar anomalias em logs de servidores Apache/Nginx.

import re

log_path = "/var/log/nginx/access.log"
pattern = r'(403|404|500)'

with open(log_path, "r") as f:
    for line in f:
        if re.search(pattern, line):
            if "union select" in line.lower():
                print(f"[ALERTA SQLi]: {line.strip()}")
